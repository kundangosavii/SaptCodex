<h1 align="center">SaptCodeX</h1>

**SaptCodeX** is a goal-driven platform designed to help engineering students become job-ready through structured roadmaps, daily tasks, and progress tracking.

## Features

* Personalized roadmap based on user goals
* Daily task execution system
* Curated resources (DSA, backend, etc.)
* Progress tracking & streak system

## Idea

Instead of overwhelming users with content, SaptCodeX focuses on **execution**—giving clear daily actions to stay consistent and improve.

## Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js

## Goal

To build a system that helps students stay consistent, track progress, and move step-by-step toward placements and real engineering skills.
